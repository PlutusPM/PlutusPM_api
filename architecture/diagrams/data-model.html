Commercial Real Estate SaaS — Data Model & Architecture

Hierarchy: Portfolio → Sites → Everything

Organization (SaaS Customer Tenant)
  └─ Portfolio (e.g., "Northeast Portfolio")
      └─ Site (CENTER: e.g., "100 Main St Tower")  ←  ALL DATA SCOPED HERE
          ├─ Buildings (N per site)
          │   └─ Floors
          │       └─ Spaces (leasable, amenity, parking)
          ├─ [ops] Assets, Work Orders, Inspections
          ├─ [tenant] Tenants, Leases, Service Requests, Reservations
          ├─ [visitor] Visitors, Visits, Passes, Access Logs
          ├─ [vendor] Vendors, Contracts, COIs
          └─ [metrics] Daily rollups, KPIs

Why site_id everywhere? RLS = can_access_site(site_id), GraphQL filter = site_id, Realtime channel = site_id, Analytics groupBy = site_id.

GraphQL + Scheduler Layer

GraphQL (pg_graphql)

Endpoint: /graphql/v1
- Auto gen from tables FKs -> nested queries
- RLS enforced: user only sees allowed site_id
- Mutations via SECURITY DEFINER functions
  comment as @graphql(type: "mutation")

Ex: opsCreateWorkOrder, 
    tenantCreateServiceRequest,
    visitorRegisterVisitor

Scheduler (pg_cron + pg_net)

pg_cron jobs (inside Postgres):
  */15 *  SLA breach detection -> notifications
  0 2 * * *  Generate PM work orders from templates
  0 8 * * *  COI expiration scan (30/60/90 days)
  0 3 * * *  Rollup metrics daily_site_stats
  0 4 * * *  Cleanup expired visitor passes
  0 9 * * 1  Lease expiration reminders

pg_net -> calls Edge Function for emails/Slack:
  select net.http_post(url:= '.../functions/v1/compliance-daily-check')

Observability:
  select * from cron.job_run_details order by start_time desc;

RLS Example (All Tables)

alter table portfolio.sites enable row level security;

create policy "access sites" on portfolio.sites for select
using (platform.can_access_site(id));

-- helper:
create function platform.can_access_site(p_site_id uuid) returns boolean
security definer as $$
  select exists (
    select 1 from platform.memberships
    where user_id = auth.uid()
    and org_id = (select org_id from portfolio.sites where id = p_site_id)
    and (site_ids is null or p_site_id = any(site_ids))
  );
$$;
